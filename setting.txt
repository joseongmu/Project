시스템 환경설정
프로그램 호환성을 위해 3명의 PC 환경설정과 서버 환경설정을 하나로 통일하는 작업을 할 것입니다. 다음과 같이 설정해주세요

Java(JDK 7.0)버전 설치

jre7/jdk_1.7.0_03으로 맞추어 주세요
제어판에서 시스템 환경변수 Path 변경해주세요

Tomcat7.0
apache-tomcat-7.0.32 버전
설치 후 
C:\Program Files (x86)\Apache Software Foundation\Tomcat 7.0
이 폴더가 있는지 확인해 주시고, Apache서버를 Start하신후
http://localhost:8080/을 쳐서 tomcat화면이 나오는지 확인해주세요.

Eclipse
www.eclipse.org 에서 Download Eclipse를 클릭하고, Java EE Developers의
window 32bit로 설치해 주세요. 이유는 Java 32bit로 설치하였으므로 Eclipse도 32bit가 아닌 경우 에러가 발생하기 때문이다.

Eclipse에서 톰캣을 서버로 등록하기

window-preference에서
server에 들어가서 Runtime Environment에 들어가 Add 버튼 클릭
Tomcat7.0과 jdk7을 지정하고 Finish하면 설정 끝
아 그리고 포트가 충돌할 경우가 있으므로 tomcat 서버가 켜져 있다면 끄도록 하자!

다음 Project Explorer 뷰에서 마우스 오른쪽 버튼 클릭하여 New로 서버를 만든다.

GowithProject0.00버전으로 해주세요.
